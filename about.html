About Us

This project aims to implement a DevOps solution for Leave Management System, an open-source application which would implemented using web-based technologies such as HTML 5, CSS 3, PHP and MySQL. The DevOps solutions that will be used in this solution are based on CI/CD principles and modern DevOps tools such as Git, Jenkins, Ansible, Nagios and AWS Cloud Platform which would run on a variety of operating systems. This proposed solution attempts to further enhance, develop and introduce automation to the existing leave management solution which will invariably enhance overall efficiency of the system.
